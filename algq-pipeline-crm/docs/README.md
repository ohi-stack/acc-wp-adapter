# Algonquian Pipeline CRM

Version: 1.0.0
Author: Onegodian | Algonquian Real Estate

## Purpose

Manages the full acquisition lifecycle from lead capture through underwriting, offer generation, contract execution, buyer assignment, funding, and closing.

## Standard Stages

- Lead Captured
- Underwriting
- Offer Sent
- Negotiation
- Under Contract
- Buyer Assigned
- Funding
- Closed
- Dead Deal

## Core Functions

- Kanban board
- Deal stage management
- Assignment tracking
- Activity logging
- Reporting
- REST API endpoints
- Automation triggers

## Integration

- Deal Intake
- MAO Engine
- Offer Generator
- Buyer Portal
- Funding Tracker
- Command Center
