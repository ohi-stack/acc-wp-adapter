# Algonquian Real Estate Platform

Production source structure for AlgonquianRealEstate.com WordPress plugins, revenue systems, transaction workflow, digital products, buyer marketplace, funding layer, document automation, e-signature, archive, analytics, and acquisitions dashboard.

## Entity Context

Algonquian Real Estate LLC is a Connecticut limited liability company organized for real estate acquisition and transaction activity. The platform should use disciplined transaction language and avoid promises of investment returns, guaranteed profits, or state-conferred authority.

## Purpose

This repository section defines and houses the WordPress-based acquisition and monetization platform for AlgonquianRealEstate.com.

The platform is designed to function as:

- Deal acquisition infrastructure
- Underwriting and offer-generation system
- Buyer marketplace
- Digital product store
- Document automation engine
- E-signature and archive system
- Funding and capital tracking system
- Education and subscription platform
- Daily acquisitions command center

## WordPress/WPBakery Rule

Always open WPBakery text blocks with:

```text
[vc_column_text]
```

Always close with:

```text
[/vc_column_text]
```

Never use malformed closing tags.

## Production Rule

A feature is not production-ready unless it is operational, documented, repeatable, permission-controlled, logged, and testable.
