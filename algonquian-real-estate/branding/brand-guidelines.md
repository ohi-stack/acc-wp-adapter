# Algonquian Real Estate Branding Guidelines

## Brand Name

Primary display name:

```text
Algonquian Real Estate
```

Legal/entity display where appropriate:

```text
Algonquian Real Estate LLC
```

Plugin author display:

```text
By Onegodian
```

Optional institutional display:

```text
By Onegodian | Algonquian Real Estate
```

## Brand Positioning

Algonquian Real Estate should be presented as a disciplined real estate acquisition, underwriting, transaction, document, and marketplace platform.

Use institutional, transaction-safe language:

- real estate acquisition platform
- transaction infrastructure
- underwriting system
- buyer marketplace
- document automation
- lender-ready records
- operational discipline

Avoid language that implies:

- guaranteed returns
- investment advice
- securities offerings
- governmental authority
- compulsory systems

## Visual Style

Primary aesthetic:

- gold
- white
- black
- charcoal
- institutional real estate style
- clean card-based layouts
- premium document-system presentation

## Plugin Card Standard

Every plugin card should include:

- logo
- plugin name
- version number
- By Onegodian
- short description
- View Details button
- Getting Started button
- Documentation button
- UI mockup image

## Logo Usage

Recommended logo placements:

- site header
- plugin library page
- command center dashboard
- PDF document headers
- buyer portal login
- digital product storefront
- document archive

## Document Header Standard

PDFs and formal documents should use:

```text
ALGONQUIAN REAL ESTATE LLC
Waterbury, Connecticut
Document Type | Deal ID | Version | Generated Date
```

## Compliance Footer

Recommended footer language:

```text
This document is generated for transaction review and internal workflow purposes. Parties should conduct independent due diligence and obtain appropriate professional advice before execution or reliance.
```
