# Plugin Specifications

## Standard Plugin Product Page

Each plugin must include:

- Logo
- UI mockup
- Version number
- By Onegodian
- View Details
- Getting Started
- Documentation
- Description
- Installation
- FAQ
- Changelog
- Screenshots
- Reviews

## Plugins

### algq-deal-intake
Captures seller leads, FSBO submissions, distressed property leads, and internal opportunities. Creates deal records and assigns IDs.

### algq-pipeline-crm
Kanban CRM for lead captured, underwriting, offer sent, under contract, buyer assigned, and closed stages.

### algq-mao-engine
Maximum Allowable Offer calculator with ARV, repairs, holding costs, closing costs, assignment fees, target profit, risk flag, and save-to-deal logic.

### algq-offer-generator
Generates purchase agreements, LOIs, cash offers, seller-financing offers, and assignment-ready documents from deal data.

### algq-buyer-portal
Buyer login, buyer dashboard, NDA gating, deal packages, bidding, buyer acceptance, and interest submissions.

### algq-funding-tracker
Tracks lenders, private capital, JV commitments, source and use, funding status, and closing conditions.

### algq-automation-engine
Trigger-action engine for status changes, document generation, emails, archive events, and closing tasks.

### algq-pdf-signature
PDF rendering, signature requests, multi-party signing, document hash, signed archive, and secure download endpoints.

### algq-document-library
Central document repository for entity, lender, acquisition, financial, compliance, and property management documents.

### algq-command-center
Daily acquisitions dashboard, KPI cards, activity feed, analytics, and document control panel.
