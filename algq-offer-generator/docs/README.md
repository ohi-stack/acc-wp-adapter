# Algonquian Offer Generator

Version: 1.0.0
Author: Onegodian | Algonquian Real Estate

## Purpose

Generates acquisition offers and transaction documents from deal records.

## Documents

- Letter of Intent (LOI)
- Purchase Agreement
- Seller Financing Offer
- Cash Offer Summary

## Automatic Page Generation

Activation creates:

- /plugin/offer-generator
- /plugin/offer-generator/start
- /plugin/offer-generator/docs
- /plugin/offer-generator/templates

## Shortcode

[algq_offer_generator]
